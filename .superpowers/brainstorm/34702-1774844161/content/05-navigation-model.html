How should the dashboard be organized?

Forget the current tabs — what's the right way to split up your player's home base?

Think of these as the top-level sections a player navigates between. Each one is a distinct "screen" within /dashboard.

A

Overview + Settings only

Two tabs. The overview does everything — tournaments, stats, activity, alts, notifications all as compact sections. Settings is the only breakout. Maximum simplicity.

Overview  |  Settings

B

Three pillars

Split by concern: what's happening, who you are, and configuration. Each is focused.

Home (activity, tournaments, actions)  |  Profile (identity, alts, stats, achievements)  |  Settings

C

Four focused tabs

More tabs, but each is tightly scoped. Nothing is overloaded.

Home (glanceable overview + actions)  |  Compete (tournaments, matches, history)  |  Profile (identity, alts, achievements)  |  Settings

D

No tabs — single scrollable page

No tab navigation at all. The dashboard is one long, well-organized page with clear section headings. Everything is visible by scrolling. Settings accessed via a gear icon or separate route, not a tab.

Like a modern profile page — scroll through your whole competitive life.
