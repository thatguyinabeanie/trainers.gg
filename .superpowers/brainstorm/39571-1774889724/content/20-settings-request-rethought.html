What changed from previous version:
  • Profile preview at top — logo + name + URL + description shown as a visual preview. Editable fields below. Shows community leaders what their community looks like publicly.
  • Tabs — Profile active, Permissions and Notifications grayed (future). Sets up for growth without needing a redesign later.
  • Single column form — instead of label/field grid. Simpler, more natural for a short form. Grid works for long forms (like player settings with 4 tabs), but community settings has ~4 fields.
  • Character count on description textarea.
  • Delete + Save on same row — delete left-aligned (destructive), save right-aligned (primary). No separate "Danger Zone" card.
  • URL not editable — shown in the preview section, not as a disabled input field.
