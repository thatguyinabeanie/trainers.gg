Alts System — Final Design Summary

Alt switcher + route structure + page designs. Ready for spec writing.

1. Route Structure

PathPageDescription
/dashboardHomeAlways aggregate overview
/dashboard/altsAll AltsAggregate stats + expandable alt list
/dashboard/alts/historyAggregate HistoryTournament history across all alts
/dashboard/alts/[username]Alt DetailPer-alt stats, teams, management
/dashboard/alts/[username]/historyAlt HistoryTournament history for specific alt
/dashboard/inboxInboxAccount-level notifications
/dashboard/settings/*SettingsAccount settings (4 sub-tabs)
/dashboard/community/[slug]/*CommunityUnaffected by alt selection
Reserved username: "history" is reserved and cannot be used as an alt username. This prevents route collision between /dashboard/alts/history (aggregate history page) and /dashboard/alts/[username] (alt detail page).

4. Alts Page — All Alts View

Route: /dashboard/alts

Layout

  • Aggregated stats at top — record, win rate, peak rating, tournament count
  • "+ New Alt" button — next to section heading
  • Alt list — each alt as a collapsible row
  • Click alt row → expands to show:
    • Per-alt stats preview (record, win rate, ELO, CP)
    • Teams preview (first 2-3 teams with sprites)
    • "View as this alt" button → changes switcher + navigates to /dashboard/alts/[username]
    • "View history" link → navigates to /dashboard/alts/[username]/history
  • Clicking does NOT auto-switch — just expands. Explicit "View as" button to switch.
  • "View all history" link → navigates to /dashboard/alts/history

5. Alts Page — Single Alt View

Route: /dashboard/alts/[username]

Layout

  • Alt profile card — large avatar (click to change), name, badges (Main, Public/Private), visibility toggle, creation date
  • Per-alt stats — record, win rate, rating (current + peak), champion points
  • Teams section — list of teams owned by this alt
    • Each team: sprite row, name, version count, tournament usage, last played
    • Actions: Open in Builder (🔨), Share (↗), Archive (📦)
    • Archived teams shown at reduced opacity
    • Click team row → opens in Team Builder
  • "View history" link → /dashboard/alts/[username]/history
  • Danger zone — delete alt (not for main alt)

6. History Pages

Aggregate History

Route: /dashboard/alts/history

Tournament history across all alts. Sortable table with alt column. Same content as current History page but at new path.

Per-Alt History

Route: /dashboard/alts/[username]/history

Tournament history filtered to specific alt. No alt column needed. Linked from alt detail page.

What changes from current implementation:
  • Sidebar "Alts & Teams" → renamed to "Alts"
  • Sidebar "History" → removed as top-level item (lives under Alts)
  • New: Alt Switcher in sidebar header (replaces logo header)
  • New route: /dashboard/alts/[username] — alt detail page
  • New route: /dashboard/alts/[username]/history — per-alt history
  • Moved route: /dashboard/history → /dashboard/alts/history
  • Reserved username: "history" cannot be an alt name
  • Switcher state derived from URL path, not client state
  • SSR/PPR/ISR compatible — all path-based, no query params
