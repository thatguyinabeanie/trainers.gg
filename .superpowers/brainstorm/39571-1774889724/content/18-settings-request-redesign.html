Design decisions:
  • Label/field grid — labels on left (200px), fields on right. Clean two-column form layout.
  • Sections separated by borders — Profile, Social Links, URL each as a section.
  • Logo upload — dashed border preview + Upload/Remove buttons. Not a clickable avatar.
  • Social links — platform prefix + input + remove button. "+ Add another" to add more.
  • URL section — read-only input showing the permanent slug.
  • Save button right-aligned — at the bottom of the form.
  • Danger Zone — red bordered card at bottom for delete action.
  • Mobile — grid collapses to single column (labels stack above fields).

Design decisions:
  • Stepped form — numbered sections (1. About, 2. Discord, 3. Social) make the form feel guided, not overwhelming.
  • Warm disclaimer — amber callout with friendly but firm language. Not a wall of bullet points.
  • Discord separated — gets its own step since it's required and reviewed. Not mixed in with optional socials.
  • Social grid — 2-column compact grid for optional accounts. Platform prefix icons.
  • Full-width submit — prominent primary button at bottom.
  • "Typically reviewed within 48 hours" — sets expectations.
  • Constrained to 560px — forms don't stretch to fill the content area.
