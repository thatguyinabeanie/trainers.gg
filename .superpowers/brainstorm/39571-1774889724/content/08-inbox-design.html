💡 Why this works: One unified feed that handles everything — notifications, actionable items, and message threads. Messages appear as collapsed thread rows (click to expand or open full conversation). The "Actions" tab surfaces items that need attention. The "Messages" tab filters to conversations only. Today we implement the notification feed + invitation actions. Tomorrow we add message threads into the same feed. No layout changes needed.

Implementation plan

Build Today

  • Notification feed with unread indicators
  • Category tabs (All, Unread, Actions, Matches, Tournaments)
  • Inline actions for invitations (Accept/Decline)
  • "Action needed" badges on urgent items
  • Mark as read / mark all read
  • Type-specific icons (match, tournament, invite, system)

Linear Ticket (Future)

  • Messages tab + message thread items in feed
  • Click thread → expand inline or open full conversation
  • Direct messaging between players
  • TO → player communications
  • Team share notifications with "View Team" action
  • AT Protocol integration for messages
  • Coach → student messaging
