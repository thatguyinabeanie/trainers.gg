How it works:
  • Group header — tournament name + update count + timestamp of latest
  • Latest notification — always shown expanded in a subtle card below the header
  • "N earlier notifications" — collapsed by default. Click to expand into compact rows.
  • Expanded earlier items — smaller text, subdued styling. "Collapse" link to close again.
  • Feed stays tight — even a tournament with 20 notifications takes up 3 lines when collapsed (header + latest + "19 earlier").
  • Same pattern works for message threads — latest message visible, older messages collapsed.
