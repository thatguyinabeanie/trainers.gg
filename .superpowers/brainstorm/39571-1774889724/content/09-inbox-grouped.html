Items at 50% opacity are future features — shown here to demonstrate how messages and team shares will slot into the same feed.

Grouping rules:
  • Tournament notifications — grouped by tournament. Shows tournament name, update count, latest item as preview. Click to expand all items.
  • Actionable items — NEVER grouped. Invitations, team submissions, check-in reminders always show as standalone items with inline action buttons + "Action needed" badge.
  • Messages (future) — grouped as conversation threads. Shows sender name, message count, latest message preview.
  • One-off notifications — org request approved, system announcements — shown as standalone items.
  • Sort order — most recent first, but actionable items float to top regardless of timestamp.
  • "Actions" tab — filters to only items with "Action needed" badge. Quick way to see what needs attention.
