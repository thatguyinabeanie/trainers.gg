Final inbox design:
  • Notifications only — no messages. Messages get their own sidebar item (future).
  • Tabs: All, Unread, Actions, Matches, Tournaments — no Messages tab.
  • Tournament notifications grouped — latest expanded, older collapsed into "N earlier."

  • One-off notifications — standalone items (org approved, judge resolved, etc.).
  • Unread indicators — blue dot on left side.
  • Type-specific icons — match (⚔), tournament (🏆), invite (📨), submission (📋), system (🔔).
