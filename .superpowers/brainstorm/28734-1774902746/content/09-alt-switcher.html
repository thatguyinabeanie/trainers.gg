Behavior

  • Closed state: Avatar + alt name + "Main alt" subtitle + ⇅ chevron
  • Dropdown: "All Alts" option at top, then list of alts with avatars, then separator + "Create new alt" + "Manage alts"
  • Current alt gets teal checkmark + subtle background highlight
  • Main badge shown on the main alt only
  • Selecting an alt → navigates to /dashboard/alts/[username]
  • Selecting "All Alts" → navigates to /dashboard/alts
  • "Create new alt" → opens create alt flow
  • "Manage alts" → navigates to /dashboard/alts
  • State derived from URL — no client state management
  • Collapsed sidebar: shows just the avatar circle, tooltip "Switch alt"
