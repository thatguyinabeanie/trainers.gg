Design decisions

  • Summary stats at top — tournaments played, win rate, best finish, avg place (respond to alt/format filters)
  • Alt filter dropdown — "All alts" default, filter to a specific alt
  • Format filter dropdown — "All formats" default, filter to VGC Reg H, etc.
  • Alt names are clickable — link to /dashboard/alts/[username]/history
  • Expandable rows — click to see team used (sprites + name), tournament summary, "View tournament →" link
  • 1st place highlight — teal color + 🏆 on first place finishes
  • Date column — short format (Mar 28)
  • Same page at /dashboard/alts/[username]/history — identical layout but pre-filtered to that alt (no Alt column needed)
