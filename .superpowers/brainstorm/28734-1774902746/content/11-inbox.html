Design decisions

  • Action items float to top — amber border + "Action needed" badge + inline buttons
  • Tournament groups — header with trophy icon + name + update count; latest expanded, older collapsed as "N earlier updates"
  • One-off notifications — standalone cards (community approved, judge resolved)
  • Unread indicator — blue dot on left
  • Read items — muted text color, no blue dot
  • Type icons — ⚔ match, 🏆 tournament, 📋 submission, ✓ approval
  • Tabs — All, Unread (with count badge), Actions (with count badge), Matches, Tournaments
  • Max-width 720px — feed doesn't stretch full page width
  • Timestamps — relative (15m ago, 2h ago, 2d ago)
