# @trainers/mobile

Expo 55 (React Native 0.83, React 19.2) mobile app. Expo Router file-based routing.

## Key Files

| File                           | Purpose                                                     |
| ------------------------------ | ----------------------------------------------------------- |
| `src/lib/api/query-factory.ts` | TanStack Query factory — use `useApiQuery`/`useApiMutation` |
| `src/lib/supabase/`            | Mobile Supabase client (SecureStore sessions)               |
| `src/app/_layout.tsx`          | Root layout, auth redirect logic                            |

## Skills

- `building-mobile-app` — Tamagui UI, navigation, data fetching
- After building features: run `checking-mobile-parity` to check for parity tickets
