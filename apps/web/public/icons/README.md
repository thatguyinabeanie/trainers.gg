# PWA Icons

Required icons for the Progressive Web App:

- `icon-192.png` (192x192) - Standard app icon
- `icon-512.png` (512x512) - Large app icon
- `icon-512-maskable.png` (512x512 with safe zone padding) - Maskable icon for adaptive icons
- `apple-touch-icon.png` (180x180) - iOS home screen icon

## Guidelines

- Generate from trainers.gg logo
- Maskable icons need 20% safe zone padding around the logo
- Use transparent background for standard icons
- Use brand background color (#1db6a5) for maskable icons
