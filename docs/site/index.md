## Landing Page

A simple, aesthetic introduction to trainers.gg. Future: dynamic content and highlights.

## Organizations

- List of organizations with logos and links
- Organization profile: logo, social links, tournaments, series highlights
- Unique, slug-based URLs for each org

## Tournaments

- List of tournaments
- Unique, slug-based URLs per tournament (scoped to org)

## Users

- Player profiles: tournaments played, results, teams, articles
- Unique, case-insensitive profile names

## Articles

- Team reports, guides, ideas, event coverage
- Tagging and recommendation system (future)

## Analytics

- Initial: data from external sources (Limitless TCG, Showdown, RK9)
- Future: in-depth analytics, metagame trends, team stats

## Dashboard

- User home: settings, preferences, profiles, teams, tournaments, analytics
- Expandable, sortable tournament tables
- Team privacy and sharing rules
- Organization creation requests
