## Resolved Issues

### 1. Alt Switcher Now Functions as Global Filter — FIXED

**Location:** `apps/web/src/components/dashboard/dashboard-sidebar.tsx` (lines 149-158, 224-242)

**Fix:** Implemented cookie-based persistence:

- `dashboard-alt` cookie (1-year expiry, path `/`, SameSite lax)
- Selecting an alt sets the cookie via `document.cookie`, then calls `router.refresh()`
- Selecting "All Alts" clears the cookie (`max-age=0`)
- Server components read the cookie via `cookies()`:
  - `dashboard/layout.tsx` (line 37) — passes `selectedAltUsername` to sidebar
  - `dashboard/page.tsx` (line 9) — filters home page data
  - `dashboard/tournaments/page.tsx` (line 21) — filters tournament data
- Cookie value validated against user's actual alts (falls back to null/"All Alts" if invalid)

This matches the spec requirement: "Global filter — selection persists across all dashboard pages via cookie (SSR-compatible)."
