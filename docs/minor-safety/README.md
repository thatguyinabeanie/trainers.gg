# 🛡️ Minor Safety & Compliance Documentation

> **⚠️ Disclaimer:** This documentation is for internal reference only and does **not** constitute legal advice. Consult a qualified attorney specializing in children's online privacy before launch.

## 📋 Overview

trainers.gg is a competitive Pokemon platform that will serve minors — VGC has Junior (under 13) and Senior (13-16) divisions. This directory contains comprehensive compliance documentation covering the major regulatory frameworks that apply to our platform.

## 🚨 Key Deadline

| Deadline | Regulation | What Happens |
|----------|-----------|--------------|
| **April 22, 2026** | COPPA 2025 Amendments | Full compliance required with new rules including expanded personal information definitions, mixed audience classification, new consent methods, and enhanced safe harbor transparency |
