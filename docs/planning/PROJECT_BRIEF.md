# Project Brief: trainers.gg - Pokemon Community Platform

## Overview

Build a comprehensive Pokemon community platform at trainers.gg that serves as a social hub for all types of Pokemon fans. The platform combines social media (powered by Bluesky AT Protocol), tournament management, team building tools, draft league organization, player profiles, and content creation in one unified experience.

## Target Audience

- VGC (Video Game Championships) competitive players
- Draft league participants
- Shiny hunters
- Casual Pokemon fans
- Content creators writing about their teams and performances
- Community organizers

## Core Features - Phase 1 (MVP)

### 1. User Profiles & Authentication

- User registration and authentication
- Trainer profiles with:
  - Display name, bio, location
  - Pokemon game preferences (VGC, Showdown, specific games)
  - Tournament history and stats
  - Team showcases
  - Social links (Twitter, YouTube, Twitch, Discord)
- Bluesky AT Protocol integration for social identity

### 2. Social Feed (Bluesky-Powered)

- Posts/updates from trainers
- Share teams, tournament results, shiny finds
- Comment and discussion threads
- Like/react to posts
- Follow other trainers
- Hashtag support for filtering (#VGC, #ShinyHunting, #DraftLeague)

### 3. Team Builder & Management (Coming Soon - Phase 1)

- Create and save Pokemon teams
- Support for VGC format (team of 6, bring 4 to battle)
- Support for other formats (Singles, Doubles, Draft)
- Team import/export (Showdown format, Pikalytics, etc.)
- Team sharing with community
- Team stats and effectiveness analysis
- Version history for team iterations

### 4. Tournament Management (Coming Soon - Phase 1)

- Create and manage tournaments
- Tournament brackets (single elimination, double elimination, Swiss)
- Registration system
- Match reporting
- Live bracket updates
- Tournament results and standings
- Player performance stats across tournaments

### 5. Draft League System (Coming Soon - Phase 1)

- Create and manage draft leagues
- Draft order and snake draft support
- Team rosters for draft leagues
- Season tracking
- Match scheduling
- League standings and stats
- Draft history and analytics

### 6. Content Creation (Articles/Blog) - Future

- Rich text editor for writing articles
- Attach teams to articles
- Tag articles by format, event, team archetype
- Author profiles
- Comments on articles
- Featured/trending articles

### 7. Community Features - Future

- Discord server integration (links, possibly widget)
- Community events calendar
- Regional/local community pages
- Meetup coordination

### 8. Search & Discovery

- Search for trainers by name, location, format
- Search teams by Pokemon, format, archetype
- Search tournaments by location, format, date
- Find draft leagues to join
- Discover articles by topic/format

## Future Features - Phase 2+

- Shiny hunting tracker with stats and collections
- Damage calculator integration
- Rental team system
- Team rating and popularity metrics
- Tournament streaming integration
- Pokemon Showdown replay embedding
- Tournament seeding based on player rankings
- ELO/ranking system
- Private messaging
- Team builder with AI suggestions

## Design Considerations

- Mobile-responsive (many Pokemon players use mobile)
- Fast performance (competitive players value speed)
- Clean, modern UI that doesn't feel too "kiddy"
- Accessibility (WCAG 2.1 AA compliance)
- Dark mode support
- Pokemon typing colors for visual hierarchy

## Success Metrics

- Active user registrations
- Teams created and shared
- Tournaments hosted
- Articles published
- Social engagement (posts, comments, follows)
- Draft leagues created
- Return user rate

## Important Notes

- Domain: trainers.gg (not yet purchased)
- Start with VGC format as primary focus, expand to other formats
- Keep trademark safety in mind - use generic terminology
- Build for scalability - Pokemon community is large
- Consider internationalization (Pokemon is global)
- Plan for Pokemon Legends Z-A and Pokemon Champions game releases in 2026

## Phase 1 Focus

Start with a solid MVP focusing on **social media features** with Bluesky integration:
- User authentication via Bluesky OAuth
- Social feed with Pokemon-curated and full Bluesky views
- User profiles
- Post creation and interactions
- Coming soon placeholders for tournaments, draft leagues, and team builder
