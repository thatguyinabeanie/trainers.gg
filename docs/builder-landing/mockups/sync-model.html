Per-team choice: keep some teams local-only on purpose

Same flow, but "local-only" is a deliberate, sticky state — a private scratchpad some teams never leave, even while logged in. More control for secret tech, but adds a concept to explain ("why is this one local and that one synced?") and a way to accidentally lose teams by clearing the browser.
