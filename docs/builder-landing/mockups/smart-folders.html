How should Smart Folders get created?

A

Save from search  ·  recommended

Run any query in the smart search (incl. predicates like incomplete, illegal, has:tyranitar), then click "Save as Smart Folder" and name it. Reuses the search we already designed — zero new query UI, and it stays live as teams change.

B

Save from search + a criteria builder

Same save-from-search, plus a small rule builder ("Format is Reg H AND has Incineroar AND missing item") for precise/compound rules that are awkward to type. More power and discoverability, but a second creation path to build and maintain.
